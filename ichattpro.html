    App details

  • Online Live Chat Application
  • Utilized JavaScript, Custom CSS, ReactJS and Socket.io to allow users functionality to send and receive text real-time.
  • Server side routing built using Express, Router, Socket.io,and cors.
  • Nodemon was used for easier server side code debugging
  • Version control was executed via Git using macOS Command Line.
  • Deployed application via Netlify for frontend and Heroku for server side code.

Tools and Tech

ReactJS, Javascript, JSX, Socket.io, Cors, NodeJS, Express, Nodemon, Git, macOS Command Line, Netlify, Heroku
